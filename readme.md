# lazycommerce 
Automate easy money.

## Getting started
- Running
  ```
  python3 manage.py migrate
  python3 manage.py runserver
  ```
- Cloning shop
  - Tokopedia
    ```
    python3 manage.py clone_shop --link https://www.tokopedia.com/laris88
    ```

## features
- clone shop
- clone goods
- automate transaction
- profit
